## Projet ITI ##

ce projet sert a generer une description pour chaque image entree. 


Le dataset utilise est "mmsample". le dataset contient 3 dossiers :  

       1) train2017, val 2017 : dossiers contenant des images pour l'entrainement et le test respectivement. 
       2) annotations : contient deux fichiers json qui decrivent les images et contiennent les labels associes.
